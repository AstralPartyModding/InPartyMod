## 一、开发者：如何生成ZIP包

### 方法：运行 package.bat

每个Mod文件夹里都有一个 `package.bat` 文件，双击运行即可自动打包。

```
📁 mods/
  └── 📁 YuGiOhCardMod/
      ├── 📄 YuGiOhCardMod.cs      ← Mod代码
      ├── 📄 package.bat            ← 🔥 双击这个！
      └── 📁 Resources/           ← Mod资源文件
```

### 打包步骤

```
┌─────────────────────────────────────────┐
│  步骤1: 双击运行 package.bat             │
│         ↓                               │
│  步骤2: 等待自动编译和打包               │
│         ↓                               │
│  步骤3: 在 dist 文件夹找到 ZIP 文件      │
└─────────────────────────────────────────┘
```

### 打包成功后的文件

```
📁 mods/YuGiOhCardMod/
  └── 📁 dist/                          ← 生成的发布包在这里
      └── 📦 YuGiOhCardMod_v0.2.0.zip   ← ✅ 这就是发布用的ZIP文件
```

### 打包脚本会做什么？

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 编译项目 | 将代码编译成DLL文件 |
| 2 | 复制资源 | 将图片、音频等资源放入正确位置 |
| 3 | 创建ZIP | 生成可直接发布的压缩包 |

## 二、玩家：如何使用ZIP包

### 核心步骤（3步搞定）

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   📦 Mod ZIP 包                                          │
│        ↓                                                │
│   【解压】到星引擎游戏目录                                │
│        ↓                                                │
│   ✅ 完成！启动游戏即可                                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 游戏目录在哪里？

通常位于：

```
Steam版本：
C:\Program Files (x86)\Steam\steamapps\common\Astral Party

或你自定义的安装路径
```

**快速找到游戏目录的方法：**

1. 打开Steam库
2. 右键点击「星引擎Party」
3. 选择「管理」→「浏览本地文件」

```
┌────────────────────────────────────────┐
│  Steam 库                              │
│  ┌────────────────────────────────┐    │
│  │ 星引擎Party                     │    │
│  │     右键 → 管理                 │    │
│  │            ↓                   │    │
│  │       浏览本地文件  ← 点击      │    │
│  └────────────────────────────────┘    │
└────────────────────────────────────────┘
```

│  ...                                    │

#### 步骤4：选择解压位置

│  ─────────────────────────               │
│  选择目标位置并提取文件                  │

  ├── 📁 Mods/                       ← Mod文件夹
  │   └── 📄 YuGiOhCardMod.dll       ← Mod程序
  ├── 📁 ModResources/               ← Mod资源
  │   └── 📁 YuGiOhCardMod/
  │       ├── 📁 cards/               ← 卡牌图片
  │       └── 📁 events/              ← 事件图片
  └── 📄 AstralParty.exe              ← 游戏主程序

## 五、常见问题解决

### ❌ 问题1：解压后找不到Mod文件夹

**现象：** 解压后游戏目录没有 `Mods` 文件夹

**解决：**

```
检查1：是否解压到了正确的游戏目录？
       应该解压到 Astral Party 文件夹内
       
检查2：ZIP包内是否包含 Mods 文件夹？
       正常ZIP结构：
       📦 Mod.zip
         ├── 📁 Mods/
         └── 📁 ModResources/
       
检查3：如果ZIP里没有Mods文件夹，手动创建：
       1. 在游戏目录新建文件夹，命名为 Mods
       2. 将 .dll 文件放入 Mods 文件夹
```

---

### ❌ 问题2：游戏启动后Mod没生效

**现象：** 控制台没有显示Mod加载信息

**排查步骤：**

```
步骤1：检查是否安装了 MelonLoader
       - 游戏目录应该有 MelonLoader 文件夹
       - 启动游戏时会弹出黑色控制台
       
步骤2：检查Mod文件位置
       正确路径：
       📁 Astral Party/
         ├── 📁 Mods/
         │   └── 📄 YuGiOhCardMod.dll  ← 应该在这里
         
步骤3：检查文件完整性
       - DLL文件是否完整（没有被杀毒软件删除）
       - 资源文件夹是否存在
```

---

### ❌ 问题3：按F1没有反应

**现象：** 游戏中按F1键没有弹出管理器

**解决：**

```
原因1：没有安装 InPartyMod 主包
解决：先安装 InPartyMod 主包（Core.dll）

原因2：键盘F1被其他软件占用
解决：尝试按 Fn+F1（笔记本可能需要）

原因3：Mod管理器未正确加载
解决：查看控制台是否有错误信息
```

---

### ❌ 问题4：解压时提示文件被占用

**现象：** 解压失败，提示文件正在使用

**解决：**

```
步骤1：关闭游戏
       确保星引擎游戏没有在运行
       
步骤2：关闭相关程序
       - 关闭文件管理器中打开的游戏目录
       - 关闭查看游戏文件的窗口
       
步骤3：重新解压
```

---

### ❌ 问题5：杀毒软件报毒

**现象：** 解压后DLL文件被杀毒软件删除

**解决：**

```
步骤1：将游戏目录添加到杀毒软件白名单
       
       Windows Defender：
       设置 → 更新和安全 → Windows安全中心
       → 病毒和威胁防护 → 排除项 → 添加文件夹
       → 选择 Astral Party 游戏目录
       
步骤2：从杀毒软件隔离区恢复文件
       
步骤3：重新解压Mod文件
```

---

### ❌ 问题6：不知道游戏安装在哪里

**快速找到游戏目录的3种方法：**

#### 方法A：通过Steam

```
Steam 库 → 右键星引擎Party → 管理 → 浏览本地文件
```

#### 方法B：通过桌面快捷方式

```
右键桌面游戏图标 → 打开文件所在位置
```

#### 方法C：通过搜索

```
按 Win+S 搜索 "Astral Party"
右键搜索结果 → 打开文件所在位置
```
